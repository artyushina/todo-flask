# Веб-менеджер задач (To-Do List) на Flask

Простое, но полноценное веб-приложение для управления списком задач с локальным запуском.  
Позволяет добавлять, отмечать выполненные, редактировать и удалять задачи. Все данные хранятся в SQLite.

---

## Возможности

- Добавление новых задач
- Отметка выполнения (чекбокс)
- Редактирование текста задачи
- Удаление задач
- Постоянное хранение в SQLite (файл `tasks.db`)

## Технологии

- **Backend:** Python 3, Flask
- **Database:** SQLite3 (встроенная)
- **Frontend:** HTML5, CSS3, JavaScript (чистый)

## Установка и запуск локально

### Требования
- Python 3.8 или выше
- pip

### Шаги

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/artyushina/todo-flask.git
   cd todo-flask
   ```

2. **Создайте и активируйте виртуальное окружение**
   ```bash
   python -m venv venv
   source venv/bin/activate    # для Linux/Mac
   venv\Scripts\activate       # для Windows
   ```

3. **Установите зависимости**
   ```bash
   pip install flask
   ```
   или
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение**
   ```bash
   python app.py
   ```

5. **Откройте в браузере**
   Перейдите по адресу [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📖 Использование

- **Добавить задачу:** введите текст в поле вверху и нажмите «Добавить».
- **Отметить выполнение:** кликните на чекбокс слева от задачи.
- **Редактировать:** нажмите кнопку с карандашом ✏️, измените текст и сохраните.
- **Удалить:** нажмите кнопку с корзиной 🗑️ (потребуется подтверждение).
